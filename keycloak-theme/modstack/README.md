# Keycloak Themes

This theme is designed to provide a clean, modern UI for [Keycloak](https://www.keycloak.org/) login, registration, and account management flows, and can be easily customized for your own deployments.

---

## ✨ Features

- Custom login and registration pages
- Clean, modern UI
- Easy to customize
- Compatible with recent Keycloak versions (tested on 22.x–26.x)
- Lightweight and production-ready

---

## 📦 Installation

1. Copy the theme directory into your Keycloak container or installation:

```bash
cp -r themes/<your-theme> /opt/keycloak/themes/
